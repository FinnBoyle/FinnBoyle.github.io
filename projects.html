Personal Projects:

  • 8x8 LED Arduino Sign with Customisable Text (2023)
    Using an Arduino Uno, three buttons and an 8x8 LED Matrix, I built an LED sign that can be customised. Buttons act as a confirm button, a move-left button and a move-right button. pressing move-left or move-right increments or decrements the currently selected list position. The list consists of the alphabet, numericals and a space, plus a final confirmation choice. Use the confirmation button to add the displayed alphanumeric to the display list, once finished, navigate to the end of the list and select the tick to display. In display mode, the system will move through the list of chosen letters, displaying each one, one at a time on repeat until the confirmation button is pressed again, where the system resets to the selection mode.
